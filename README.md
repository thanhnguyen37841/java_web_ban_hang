## 🛒 Website Quản lý và Bán Văn phòng phẩm 

**Vai trò:**  phát triển Backend (Phân hệ Admin)

### Mô tả
Tham gia phát triển website thương mại điện tử theo mô hình MVC sử dụng Java Spring Boot.

### Phụ trách
* Quản lý đơn hàng và xử lý hoàn trả tồn kho khi hủy đơn.
* Quản lý tài khoản người dùng, hỗ trợ khóa/mở khóa tài khoản.
* Quản lý mã giảm giá (Coupon): CRUD, kích hoạt/vô hiệu hóa, kiểm tra điều kiện áp dụng.
* Quản lý và kiểm duyệt đánh giá sản phẩm.
* Xây dựng API và xử lý nghiệp vụ với Spring Boot, Spring Data JPA và Spring Security.

### Công nghệ sử dụng
* Java
* Spring Boot
* Spring Security
* Spring Data JPA (Hibernate)
* Thymeleaf
* Bootstrap 5
* MySQL
