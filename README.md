Website Quản lý và Bán Văn phòng phẩm

Công nghệ: Java • Spring Boot • Spring Security • Hibernate • Thymeleaf • Bootstrap • MySQL


Tham gia phát triển hệ thống thương mại điện tử theo mô hình MVC.
Phụ trách xây dựng các chức năng quản trị gồm:
Quản lý đơn hàng và xử lý hoàn kho khi hủy đơn.
Quản lý tài khoản người dùng và khóa/mở khóa tài khoản.
Quản lý mã giảm giá (CRUD, kích hoạt/vô hiệu hóa, kiểm tra điều kiện áp dụng).
Quản lý và kiểm duyệt đánh giá sản phẩm.
Thiết kế API, xử lý nghiệp vụ bằng Spring Boot và Spring Data JPA, tích hợp Spring Security để phân quyền và xác thực.
