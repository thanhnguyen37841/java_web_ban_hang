## 🛒 Website Quản lý và Bán Văn phòng phẩm 

**Vai trò:**  Backend Developer (Phân hệ Admin)

### Mô tả
Tham gia phát triển website thương mại điện tử theo mô hình MVC sử dụng Java Spring Boot.

### Phụ trách
* Phát triển chức năng quản lý đơn hàng và xử lý hoàn trả tồn kho khi đơn hàng bị hủy.
* Xây dựng chức năng quản lý tài khoản người dùng, hỗ trợ khóa/mở khóa tài khoản.
* Phát triển chức năng quản lý mã giảm giá (Coupon): CRUD, kích hoạt/vô hiệu hóa và kiểm tra điều kiện áp dụng.
* Xây dựng chức năng quản lý và kiểm duyệt đánh giá sản phẩm.
* Thiết kế API và xử lý nghiệp vụ bằng Spring Boot, Spring Data JPA (Hibernate) và Spring Security.

### Công nghệ sử dụng
* Java
* Spring Boot
* Spring Security
* Spring Data JPA (Hibernate)
* Thymeleaf
* Bootstrap 5
* MySQL
